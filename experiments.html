Mammal March Madness authenticated REST API (in progress)

A RESTful API with JWT authentication built with .NET Core 2.2 project for a future web app implementation of Mammal March Madness, an alternative to basketball for the science-minded. More information here and here

See the source code on Github:


https://github.com/cbrenberg/mammal-march-madness-api

Dungeons and Dragons Slackbot

A distributed Slackbot application running on Heroku and AWS Lambda using Node, Express, GraphQL, and Wit.ai natural language processing to provide access to Spell information from the 5th Edition Dungeons and Dragons API found at http://www.dnd5eapi.co/

See the source code on Github:

  • https://github.com/cbrenberg/nlp-slackbot
  • https://github.com/cbrenberg/serverless-spell-service
  • https://github.com/cbrenberg/serverless-dnd-graphql
